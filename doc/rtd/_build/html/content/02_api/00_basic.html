Wrapper Root Class

Ver. 2.0.0 (2023-05-12)

This module provides wrapper classes to embed River functionalities into MLPro. Currently, the following topics are supported by the wrapper:

  • Native data streams

  • Selected cluster algorithms

Learn more: https://www.riverml.xyz/
